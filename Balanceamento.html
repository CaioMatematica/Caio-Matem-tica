Conceito

O balanceamento de equações químicas consiste em ajustar os coeficientes de modo que o número de átomos de cada elemento na equação química seja conservado.

Passo a passo de como fazer

Primeiro deve-se contar a quantidade de cada elemento de uma equação.

Após isso deve-se mudar os coeficientes dos elementos sem mecher no índice
